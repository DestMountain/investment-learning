## 🎯 学习路线

1. **基础入门** → 了解投资品种和市场规则
2. **技术分析** → 读懂 K 线，掌握常用指标
3. **基本面分析** → 学会看财报，判断公司价值
4. **策略构建** → 打造自己的交易系统
5. **风险管理** → 活下来，才能赚钱
6. **心理修炼** → 克服人性的弱点
